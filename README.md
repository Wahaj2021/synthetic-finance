# Synthetic Finance Data Analysis (Python)

##  Business Problem

Financial institutions rely on data analysis to understand revenue patterns, customer financial behaviour, and risk indicators.

This project explores a synthetic financial dataset to identify trends, patterns, and potential business insights using Python.

---

##  Objectives

- Perform exploratory data analysis (EDA)
- Identify key financial trends
- Analyse relationships between financial variables
- Detect potential risk or performance indicators
- Provide business-focused recommendations

---

##  Tools Used

- **Python**
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

##  Dataset Overview

The dataset contains synthetic financial data including:

- Customer financial attributes
- Account balances
- Transaction-related metrics
- Risk-related indicators (if applicable)

> Dataset created for analytical and portfolio demonstration purposes.

---

##  Data Cleaning & Preparation

- Checked for missing values
- Removed duplicate records
- Standardised column names
- Converted relevant columns to appropriate data types
- Created derived features where necessary

---

##  Key Analyses

- Distribution of financial metrics
- Correlation analysis between variables
- Risk-related pattern identification
- Trend exploration across segments

---

##  Key Insights

- Certain financial indicators show strong relationships with risk/performance variables.
- Distribution patterns highlight potential high-value or high-risk segments.
- Correlation analysis reveals important drivers influencing financial outcomes.

---

##  Business Recommendations

- Monitor high-risk financial indicators proactively.
- Focus on high-value customer segments for revenue optimisation.
- Use predictive modelling in future work to improve financial risk assessment.
